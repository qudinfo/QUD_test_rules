# Инструменты и хеджирование

## Разрешённые инструменты
- Допускаются основные валютные пары (**Forex majors**) и металлы **XAUUSD**, **XAGUSD**.
- Инструменты с биржевыми/брокерскими суффиксами могут быть запрещены; используйте базовые тикеры.

## Запрет хеджирования
- **Хеджирование на одном инструменте запрещено** (одновременные разнонаправленные позиции по одному символу).
- Выявление хеджа ⇒ позиция может быть закрыта, нарушение фиксируется.

## Запрещённые инструменты
- Позиции по тикерам вне списка разрешённых могут быть закрыты, нарушение фиксируется.
