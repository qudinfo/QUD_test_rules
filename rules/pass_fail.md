# Критерии удачного теста PASS / FAIL

Итоговый статус формируется по результатам периода (7 дней) демо-теста при отсутствии нарушений правил.

| Статус   | Условие результата |
|---|---|
| **PASS-A** | Итоговая доходность **≥ +4%**, без нарушений |
| **PASS-B** | Итоговая доходность от **0% до –2%**, без нарушений |
| **FAIL**   | Любое нарушение лимитов рисков / политики SL / запрета хеджирования / списка инструментов; либо < **3** завершённых сделок |

> По окончанию тестового периода все вновь отркытые сделки будут блокироваться

**Минимальная торговая активность:** не менее **3** завершённых сделок за период теста.
